## Usage

### Run the app locally

Now you should see the project at [http://localhost:3000](http://localhost:3000)

```bash
yarn dev
```

### Build the app

This will output the production files to the `/dist` folder.

```bash
yarn static
```

### Deploy to production

```bash
yarn deploy:prod
```
